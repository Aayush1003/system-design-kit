# 🚀 SysDesign.AI - Premium System Design Kit

A state-of-the-art interactive kit for mastering system design, built with AI-inspired aesthetics and modern web technologies.

## ✨ Features

- **Fundamentals**: Deep dives into scalability, availability, and reliability.
- **Component Library**: Visual building blocks for modern architectures.
- **Design Patterns**: Interactive guides for Sharding, Microservices, and more.
- **Interactive Notes Drawer**: Translucent slide-out sheets for technical deep-dives, architectural trade-offs, and typical interview Q&As.
- **Dynamic CRUD Editor**: Edit any system note inline or create completely new custom note cards with direct markdown support.
- **Data Operations & Backup**: Effortlessly export your custom notes library to a JSON file or import backups in one click.
- **Case Studies**: Real-world architectural breakdowns (YouTube, Uber) with SVG diagrams whose nodes dynamically link back to detailed concept sheets!
- **AI Architect**: An AI-powered assistant to help synthesize production-grade designs.
- **Automated Free Hosting**: Pre-configured CI/CD workflow to host the compiled application on GitHub Pages for free on every push.

## 🛠️ Technology Stack

- **React 18** + **Vite**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Vanilla CSS** (Custom Design System)

## 🚦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

This kit uses a **Dark Mode First** approach with:
- **Glassmorphism**: Translucent panels for a modern look.
- **Micro-interactions**: Smooth hover effects and layout transitions.
- **Responsive Layout**: Optimized for both desktop and tablet views.
